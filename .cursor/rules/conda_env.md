# Agent configuration

## Python environment

This project uses a Conda environment named `AI-Engg-HTX`.

You must activate this environment at the start of every terminal session with the command:

`conda activate AI-Engg-HTX`

All Python commands must use this conda environment interpreter.

Do NOT use `python` or `pip` until `AI-Engg-HTX` is activated.

To simultaneously check which environment is active and which environments are available, run this command:

`conda env list`
